.. only:: latex or latexpdf or text

    Below are thumbnails with links to Jupyter Notebook examples
    for the IDAES Process Systems Engineering (IDAES-PSE) toolkit.

    `IDAES-PSE project <https://idaes-pse.readthedocs.io/>`_.
